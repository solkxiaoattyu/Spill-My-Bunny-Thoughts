# 本地存储字段说明（浏览器 localStorage / sessionStorage）

本项目**无独立后端数据库**，用户数据保存在浏览器本地。语料库为静态 JSON 文件。

## 语料库（静态数据）

| 文件 | 说明 |
|------|------|
| `corpus/tagged_corpus.json` | 主语料库，约 3468 条已打标文案，前端运行时加载 |
| `corpus/tagged_corpus.csv` | 同上，CSV 格式，便于 Excel / 数据分析 |
| `corpus/03_朋友圈文案清洗结果.csv` | 打标前的原始清洗文案 |
| `corpus/qc_report.json` | 语料质检报告 |
| `标签枚举/tags_enum.json` | mood / scene / style 等标签 ID 与中文名映射 |

### 单条语料结构（JSON）

```json
{
  "id": 1,
  "text": "文案正文",
  "tags": {
    "mood": ["mood_happy"],
    "scene": ["scene_daily"],
    "style": ["style_minimal"],
    "purpose": ["purpose_share_life"],
    "theme": ["theme_happiness"],
    "length": "length_short",
    "keywords": ["关键词"]
  },
  "qc_passed": true
}
```

## localStorage 键名

| 键名 | 用途 | 定义位置 |
|------|------|----------|
| `favorite-copies` | 收藏文案列表 | `src/data/matchQuery.ts` |
| `copied-copies` | 复制记录 | `src/data/matchQuery.ts` |
| `browse-history` | 浏览历史 | `src/data/matchQuery.ts` |
| `custom-match-query` | 最近一次 AI/本地匹配输入 | `src/data/matchQuery.ts` |
| `custom-match-results` | 最近一次匹配结果 | `src/data/matchQuery.ts` |
| `match-history` | 匹配历史 | `src/data/matchHistory.ts` |
| `random-pick-history` | 随机抽取历史 | `src/data/randomPickHistory.ts` |
| `quiz-matched-results` | 答题匹配结果 | `src/data/quiz.ts` |
| `quiz-answers` | 答题选项 | `src/data/quiz.ts` |
| `ai-api-config` | AI API 配置（Key / 模型 / 开关） | `src/data/aiConfig.ts` |
| `yw-daily-history` | 今日推荐已推送 ID（近 30 天） | `src/services/chatchat/corpusStorage.ts` |
| `isLoggedIn` | 登录状态（演示用） | `src/pages/Login.tsx` |
| `userPhone` | 登录手机号（演示用） | `src/pages/Login.tsx` |

## sessionStorage 键名
