# 06 · 前端页面与组件

> 文档目的：把 9 个页面 + 21 个组件的职责、状态、样式约定、持久化讲清楚，方便定位 UI 问题。
> 涉及文件：`src/{App,pages,components,context,hooks}/*`、`src/index.css`。

---

## 1. 全局结构

### 1.1 `App.tsx`

- 最大宽度 430px 容器，模拟小程序
- `<Routes>` 11 条路由（见 `01_项目总览` §4）
- 顶层挂 `<CorpusProvider>`（在 `main.tsx`）、`<CopyToast>`、`<Welcome>` 闪屏

### 1.2 `CorpusContext`

App 启动时 `Promise.all([loadCorpus(), loadCorpusIndex()])`，全局暴露：
```ts
{ status: "loading" | "ready" | "error", error: string | null, count: number }
```
所有需要语料的页面用 `useCorpus()` 拿状态，未 ready 时禁用匹配按钮。

## 2. 页面职责

### 2.1 `Home.tsx`（主页）

- 顶部：朋友圈风格 header（`HomeMomentsHeader`）+ 用户简介（`HomeProfileBio`）
- 主卡片（`HomeMainCard`）：输入「这一刻想法」→ 匹配 → 弹 `MatchedCopyModal`
- AI 状态条（`HomeApiStatusBar`）：显示当前 AI 模式 / 配置入口
- 今日推荐（`StackedDailyRecommend`）：3 条语料随机抽，每日固定
- 设置抽屉（`ProfileSettingsSheet`）：AI 配置

### 2.2 `Quiz.tsx` / `QuizResults.tsx`

- `Quiz`：问卷入口页，按钮打开 `QuizModal`
- `QuizResults`：展示 `QuizResultsModal`（从 localStorage 读上次结果）

### 2.3 `RandomPick.tsx`

- 随机抽文案盲盒（`RandomPickModal`），3 条一批
- 抽盒历史入口

### 2.4 `Profile.tsx`（我的）

- 头像 + 昵称（`AppAvatar`）
- Bento 网格入口：浏览历史 / 收藏 / 已复制 / 抽盒历史 / 匹配历史 / AI 设置
- 统计数字（收藏数、复制数等）

### 2.5 历史类页面

| 页面 | 数据源 | localStorage key |
|------|--------|-----------------|
| `BrowseHistory` | 浏览历史 | `browse-history` |
| `Favorites` | 收藏 | `favorite-copies` |
| `CopiedHistory` | 已复制 | `copied-copies` |
| `DrawHistory` | 抽盒历史 | `random-pick-history` |
| `MatchHistory` | 匹配历史 | `match-history` |

### 2.6 `Login.tsx` / `Welcome.tsx`

- `Login`：占位登录页（手机号输入，无真实鉴权）
- `Welcome`：启动闪屏，3 秒后或点击跳过 → `/home`

## 4. 状态与持久化

### 4.1 全局状态

- `CorpusContext`：语料加载状态（见 §1.2）
- 其余全部用组件内 `useState`，跨页用 localStorage

### 4.2 localStorage key 清单

| key | 内容 | 写入处 |
|-----|------|--------|
| `quiz-answers` | 问卷答案 6 槽位 | `QuizModal` 提交时 |
| `quiz-matched-results` | 问卷匹配结果 | `QuizModal` / `QuizResultsModal` |
| `quiz-refresh-remaining` | 剩余刷新次数 | 同上 |
| `yourword-quiz-match-session` | 问卷刷新会话（含 baseline） | `matchFromQuiz` |
| `custom-match-query` | query 模式上次查询 | `Home.handleMatch` |
| `custom-match-results` | query 模式上次结果 | 同上 |
| `yourword-query-match-session` | query 刷新会话 | `runQueryMatch` |
| `yourword-random-pick-session` | 随机抽盒会话 | `RandomPickModal` |
| `yw-daily-history` | 今日推荐 30 天历史 | `corpusStorage` |
| `match-history` | 匹配历史 30 条 | `matchHistory` |
| `browse-history` | 浏览历史 50 条 | `matchQuery.recordBrowseHistory` |
| `favorite-copies` | 收藏 | `matchQuery.toggleFavoriteCopy` |
| `copied-copies` | 已复制 50 条 | `matchQuery.recordCopiedCopy` |
| `random-pick-history` | 抽盒历史 | `randomPickHistory` |
| AI 配置相关 | apiKey/baseUrl/model | `aiConfig` |

### 4.3 自定义事件

- `yourword-copy-success`：复制成功时 dispatch，`CopyToast` 监听
- `yourword-reduce-recommend`：减少推荐时 dispatch

## 5. 样式约定

### 5.1 全部在 `src/index.css`（约 3400 行）

- 无 CSS Module / Tailwind / styled-components
- BEM 风格命名：`quiz-sheet-section`、`quiz-tag--active`
- 手绘 SVG icon（`SketchMood` / `SketchScene` 等）配合 `.sketch-stroke`

### 5.2 设计语言

- 黑白为主 + 品牌色点缀（`fill-brand` / `fill-brand-pink`）
- 圆角 12~16px，卡片化
- 手绘感线条（`sketch-stroke` 类 + `stroke-width 1.6~1.8`）
- 动画：`cubic-bezier(0.34, 1.28, 0.64, 1)` 弹性入场

### 5.3 关键 class 前缀

| 前缀 | 作用域 |
|------|--------|
| `quiz-sheet-*` | 问卷弹窗 |
| `quiz-tag-*` | 问卷选项按钮 |
| `quiz-preset-*` | 懒人推荐卡 |
| `quiz-results-sheet*` | 结果弹窗 |
| `profile-bento-*` | 我的页 Bento 网格 |
| `moments-*` | 朋友圈预览 |

### 5.4 后续可优化

- [ ] `index.css` 单文件 3400 行，组件增多建议拆 CSS Module 或 vanilla-extract
- [ ] 主题色用 CSS 变量统一管理（当前硬编码 `#111` / `#888` 等）
- [ ] 暗色模式未支持
- [ ] 移动端安全区域 `env(safe-area-inset-*)` 未处理

## 6. hooks

### 6.1 `useCopyMeta(text)`

根据文案文本反查语料，返回 `{ tagLabel, copyId }`，用于历史页面展示标签徽章。

### 6.2 `useQuoteActions(text)`

封装单条文案的操作：复制 / 收藏切换 / 翻译 / 浏览历史记录。
内部组合 `copyTextToClipboard` / `toggleFavoriteCopy` / `translateQuoteSemantically` / `recordBrowseHistory`。
