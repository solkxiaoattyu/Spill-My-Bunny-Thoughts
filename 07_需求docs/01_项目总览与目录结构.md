# 01 · 项目总览与目录结构

> 文档目的：让你和我对整个小程序「想到一句话」的代码骨架有一份共同地图，方便后续优化细节时快速定位。
> 本文件会随重构更新；遇到与代码不一致时以代码为准，并请提醒我同步本文件。

---

## 1. 产品一句话

一个帮用户「找到一条合适的朋友圈文案」的小程序式 Web App。
输入有两种方式：

- **快速匹配（问卷）**：5 步选择题 + 懒人推荐卡 → 输出 3 条文案。
- **这一刻想法（自然句）**：用户写一句话 → AI / 离线向量语义匹配 → 输出 3 条文案。

输出后可复制、收藏、看翻译、预览朋友圈效果、换一批、看匹配历史。

## 2. 技术栈


| 层   | 选型                           | 备注                              |
| --- | ---------------------------- | ------------------------------- |
| 框架  | React 18 + TypeScript        | 单页应用，最大宽度 430px 模拟小程序           |
| 构建  | Vite 6                       | `npm run dev` / `npm run build` |
| 路由  | react-router-dom v6          | 见 `App.tsx`                     |
| 样式  | 原生 CSS + 手绘 SVG              | 全部在 `src/index.css`，无 UI 库      |
| 图标  | lucide-react                 | 仅少量                             |
| 状态  | React Context + localStorage | 无 Redux / Zustand               |
| 后端  | 无                            | 纯前端，AI 走用户自配 OpenAI 兼容接口        |
| 语料  | 静态 JSON                      | 打包时进 dist，运行时 fetch             |
